# Garbage-Classification
Waste image classification using deep learning (ANN &amp; CNN).

## Project Overview
This project develops an automated waste classification system using Deep Learning. Artificial Neural Networks (ANN) and Convolutional Neural Networks (CNN) are applied to classify garbage images into predefined categories, aiming to improve accuracy and support more efficient waste sorting.
